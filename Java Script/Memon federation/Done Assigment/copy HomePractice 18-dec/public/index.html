Push = Add something in ending of array. Method = variablename.push("something");

Pop = Remove something in ending of array. Method = variablename.pop("something");

Unshift = Add something in starting of array. Method = variablename.unshift("something");

Shift = Remove something in starting of array. Method = variablename.shift("something");

Splice = Use the splice method to insert one or more elements anywhere in an array, while optionally removing one or more elements that come after it. Suppose you have an array with the elements "dog", "cat", "fly", "bug", "ox". The following code adds "pig", "duck", and "emu" after "cat" while removing "fly" and "bug" pets.splice(2, 2, "pig", "duck", "emu");

Slice = Copy something in new array. Method = var newVariablename=variablename.slice(2, 6);
